How to install mods - Rules

  1. Tym is always right
  2. Be friendly and kind to each others
  3. use common sense
  4. When Tym is wrong, refer to rule #1
  1. Download mods
  2. Install forge for 1.12.2 with recommended build - installer
  3. Run forge installer, select client
  4. Open Minecraft launcher > installations tab
  5. Click "New/Nouveau"
  6. Select version "release-1.12.2-forge-XXXXXX"
  7. Next to play/jouer button, select installations you created
  8. Run minecraft, close once on menu screen
  9. Open file explorer, in URL address bar type: %APPDATA%
  10. Go to ".minecraft" folder, create "mods" folder, and "1.12.2" folder inside "mods"
  11. Paste *.jar files inside ".minecraft/mods/1.12.2"
  12. Open minecraft again
  13. Play :-)
